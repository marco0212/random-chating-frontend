# Random chat App frontend

React.JS와 Socket.IO를 사용해 구현한 랜덤 채팅 어플리케이션입니다.

## Features

- Set nickname
- Find peer to chat
- Find other peer until you're satisfied

## Skills

- React
- Redux
- socket.io-client
- Styled component

## Install

**Install**
using yarn: `yarn install`
Or npm: `npm install`

**Execute**
using yarn: `yarn start`
Or npm: `npm start`

**test**
using yarn: `yarn test`
Or npm: `npm test`

### 테스트 후기

한동안 백엔드만 만지다가 오랜만에 리엑트 작업을 하려니 기억이 가물가물해서 쉽지 않았다. 첫째 날, 당연히 될거라고 생각했던 CRA이 버전관련 이슈로 제대로 설치되지 않았을 때 식은 땀이 흐르고 마음이 콩닥콩닥거렸다. 테스트를 하며 가장 힘들었던 것은 아무래도 주어진 시간내에 만들어야 한다는 조급한 마음을 컨트롤하는 일이었던 거 같다.

또 실시간 채팅을 구현하기 위한 솔루션으로 socket.io를 찾게 되었는데 새로운 것을 공부하며 구현하는 것이 힘들었다. 오래전에 보기만하고 넘어갔었는데 그때가 원망스러웠고 문서를 읽고 찾아보는데 꽤 많은 시간을 사용했던 거 같다.

현재 요구사항은 만족되어 있지만 텍스트 전달 이외에 이미지나 영상도 주고 받을 수 있게 만들면 재밌을 거 같다는 생각이 든다.
